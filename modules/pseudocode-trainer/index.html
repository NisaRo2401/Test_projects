IHK Pseudocode – Syntax-Referenz

Variablen & Zuweisung

// Deklaration zahl : GANZZAHL // auch: INTEGER name : ZEICHENKETTE // auch: STRING flag : WAHRHEITSWERT // auch: BOOLEAN preis : KOMMAZAHL // auch: REAL, FLOAT // Zuweisung zahl ← 42 // oder: zahl := 42

Ein- & Ausgabe

EINGABE zahl // auch: INPUT(zahl) AUSGABE zahl // auch: OUTPUT(zahl), PRINT zahl

Fallunterscheidung

WENN bedingung DANN // oder: IF ... THEN anweisung SONST WENN andere_bed DANN // ELSE IF anweisung SONST // ELSE anweisung ENDE WENN // END IF

FOR-Schleife (Zählschleife)

FÜR i ← 1 BIS 10 MACHE // FOR i ← 1 TO 10 DO anweisung ENDE FÜR // END FOR // Mit Schrittweite: FÜR i ← 10 BIS 1 SCHRITT -1 MACHE ... ENDE FÜR

WHILE-Schleife (kopfgesteuert)

SOLANGE bedingung WIEDERHOLE // WHILE ... DO anweisung ENDE SOLANGE // END WHILE

REPEAT-UNTIL (fußgesteuert)

WIEDERHOLE // REPEAT anweisung BIS bedingung // UNTIL ...

Arrays (Felder)

feld[0..9] : GANZZAHL // Array mit 10 Elementen feld[0] ← 42 // Schreiben x ← feld[i] // Lesen n ← LÄNGE(feld) // Anzahl Elemente

Operatoren

// Arithmetisch + - * / MOD DIV // MOD = Rest, DIV = ganzzahl. Division // Vergleich = ≠ < > ≤ ≥ // oder: == != <= >= // Logisch UND ODER NICHT // AND OR NOT

Funktion & Prozedur

FUNKTION name(param : TYP) : RÜCKGABETYP ... RÜCKGABE wert // RETURN wert ENDE FUNKTION PROZEDUR name(param : TYP) ... ENDE PROZEDUR
